# Football Squad Builder v9

## New in v9: data persistence

The Vite React app now saves these values to browser `localStorage`:

- registered players
- captain selection
- selected formation
- custom formation slots and dragged positions

So when you refresh the page, your squad data stays.

There is also a **Clear Saved Data** button to reset everything.

Run:

```bash
npm install
npm run dev
```

Open:

```txt
http://localhost:5173
```
